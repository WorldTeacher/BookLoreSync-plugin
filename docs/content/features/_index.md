# Features

This section explains each major feature area of the plugin in detail.

| Feature | Description |
|---------|-------------|
| [Session Tracking](@/features/session-tracking.md) | How reading sessions are detected, validated, and stored |
| [Offline Support](@/features/offline-support.md) | Local queue, retry logic, and batch upload |
| [Ratings](@/features/ratings.md) | Syncing book ratings from KOReader to BookLore |
| [Annotations](@/features/annotations.md) | Syncing highlights and notes, colour mapping, EPUB CFI |
| [Auto-Update](@/features/auto-update.md) | Keeping the plugin up to date from within KOReader |
| [Gestures and Buttons](@/features/gestures.md) | Assigning plugin actions to gestures, tap zones, and hardware buttons |

For configuration of these features, see the [Configuration](@/configuration/_index.md) section.
